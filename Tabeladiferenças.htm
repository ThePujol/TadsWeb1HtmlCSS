Diferenças entre HTML e HTML5

Funcionalidade HTML HTML5
Definição de tipo de documento <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> <!DOCTYPE html>
Meta charset <meta http-equiv="Content-Type" content="text/html; charset=UTF-8"> <meta charset="UTF-8">
Semântica Estruturação usando <div> e <span> com classes para identificar elementos. Introdução de novas tags semânticas como <header>, <nav>, <section>, <footer>, etc.
Reprodução de mídia Dependência de plugins (como Flash) para reproduzir vídeos e áudios. Introdução das tags <video> e <audio> para reprodução nativa de vídeos e áudios.
Armazenamento local Utilização de cookies ou armazenamento em sessão do lado do servidor. Introdução do armazenamento local usando localStorage e sessionStorage.
Formulários Limitações nos tipos de input e validações complexas. Introdução de novos tipos de input, validações nativas e API para formulários mais avançados.
Canvas Não suportado. Introdução do elemento <canvas> para desenhar gráficos, animações, etc.
